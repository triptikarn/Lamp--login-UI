# ✨ Futuristic Lamp Login UI

An animated and interactive frontend login page built with modern web technologies and creative UI effects.

## 💡 Project Description

This project features a cute interactive SVG lamp animation connected with a futuristic login interface.

When the user pulls the lamp cord:

* The lamp turns ON/OFF
* The login form appears/disappears
* Dynamic glow colors change in real time

The background includes:

* Animated stars
* Nebula lighting effects
* Floating gradient blobs
* Cyber-style grid animations

## 🚀 Technologies Used

* HTML5
* CSS3
* JavaScript
* GSAP Animation Library
* GSAP Draggable Plugin
* SVG Animations

## 🎨 UI Features

* Glassmorphism design
* Smooth animations and transitions
* Interactive user experience
* Modern futuristic styling
* Responsive layout

## 📚 Learning Experience

This is an AI-assisted frontend project created to explore:

* Advanced UI animations
* Interactive frontend design
* SVG manipulation
* GSAP animation workflows
